What the CSS?

Cascading Style Sheets (CSS) is used to format the lauout of a webpage

With CSS, you can controlthe color, font, the size of text, the spacing between elements, hopw elements are positioned and laid out, what background images or background colors are to be used, different devices and screen sizes, anh much more!

Tip:The word cascadingmeans that a style applied to a parent element will also apply to all children elments within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and other text elements within the body will also get the same color (unless you specify something elsse)
